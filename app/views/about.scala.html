About IrfanView

IrfanView is a very fast, small, compact and innovative FREEWARE (for non-commercial use) graphic viewer for Windows 9x, ME, NT, 2000, XP, 2003 , 2008, Vista, Windows 7, Windows 8.
It is designed to be simple for beginners and powerful for professionals, and seeks to create unique, new and interesting features.

IrfanView was the first graphic viewer with Multiple ICO support. The first Windows graphic viewer with Multiple (animated) GIF support. Among the first graphic viewers with Multipage TIF support.
