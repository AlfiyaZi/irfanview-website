How to install additional languages for IrfanView?

  1. Download language file (see below)
  2. Unzip the downloaded file into your "IrfanView\Languages\" folder
  3. Start IrfanView and go to "Properties->Language"

Please use always latest IrfanView version and latest language versions. Keep in mind that US English is always included.

A huge THANKS to all users/translators sending me new languages and updates!
