How to install additional toolbar skins for IrfanView 4.00 or later?

  1. Download toolbar ZIP file (click on the toolbar image below)
  2. Unzip the downloaded file into your "IrfanView\Toolbars\" folder
  3. Start IrfanView and go to "Properties->Toolbars"

Designers: if you create a new toolbar skin, please send me an email with the PNG/TXT files.

A huge THANKS to all users sending me icons and toolbars during last years!
